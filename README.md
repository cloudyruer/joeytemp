# joeytemp

## Install

```
npm install -g https://github.com/cloudyruer/joeytemp.git
```

## Usage

### Try This First =)

```
joeytemp createFast
```

### Default:

```
joeytemp folderName
```

| folderName  |
| :---------: |
| index.html  |
|  style.css  |
|  script.js  |
| .prettierrc |
| .gitignore  |

### folderName -one:

```
joeytemp folderName -one
```

| folderName |
| :--------: |
| index.html |

### folderName -jq:

```
joeytemp folderName -jq

```

|   folderName    |
| :-------------: |
|   index.html    |
|    style.css    |
| jquery.js (3.6) |
|    script.js    |
|   .prettierrc   |
|   .gitignore    |

### folderName -bs:

```
joeytemp folderName -bs
```
